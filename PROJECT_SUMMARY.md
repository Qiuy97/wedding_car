## 核心业务流程

### 用户预订流程
```
首页浏览 → 搜索筛选 → 车辆详情 → 立即预订 → 填写信息 → 确认订单 → 微信支付 → 预订成功 → 订单管理
```

### 车主服务流程  
```
车辆注册 → 信息完善 → 接收订单 → 确认服务 → 提供服务 → 服务完成 → 收入到账 → 用户评价
```
